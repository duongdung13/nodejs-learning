Mock Question 1.4
Time: 30 min • Score: 30
Type: Code Implementation

Full Description:
Bạn cần xây dựng một REST API đơn giản cho phép người dùng upload một file ảnh. API phải xác thực loại file hợp lệ (chỉ cho phép JPEG, PNG), lưu file lên thư mục uploads/ và trả về URL để truy cập ảnh.

Yêu cầu:

Endpoint: POST /upload

File gửi lên thông qua form-data với field name là image

Validate: file không lớn hơn 5MB và đúng định dạng JPEG hoặc PNG

Lưu file vào thư mục uploads/ (tạo thư mục nếu chưa có)

Trả về JSON: { url: "/uploads/filename.png" }

Giới hạn:

Chỉ dùng core module (không dùng express, multer,...)
