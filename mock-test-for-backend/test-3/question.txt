Time: 30 min • Score: 30
Type: Code Implementation

Full Description:
Viết một HTTP server có endpoint /proxy?url=https://... để thực hiện role proxy: nhận request GET đến /proxy, lấy nội dung từ URL chỉ định và trả lại response đó cho client.

Yêu cầu:

Validate URL hợp lệ và bắt đầu bằng http hoặc https

Sử dụng module http và https để gửi request

Nếu URL trả về lỗi (404, 500...), phản hồi lỗi tương ứng

Xử lý lỗi timeout, mạng

Giới hạn:
Không dùng axios hoặc fetch. Chỉ dùng core modules.
